Detailed Description

uptime counter takes a hardware counter and calculates the uptime in nanoseconds.

This class should be used over calling frequency::duration_from_cycles() to calculate uptime because that function will overflow when 9223372 seconds or around 15 weeks have elapsed. This has to do with how the integer arithmetic is handled for converting from frequency to duration using a cycle count.

So long as this class's uptime() is at least once every 15 weeks, the uptime will be accurate up to ~292 years.
