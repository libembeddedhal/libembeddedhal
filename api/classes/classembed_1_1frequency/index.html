        embed::frequency

        embed::frequency Class Reference

        Represents the frequency of a signal. It consists of a single integer 64-bit number that presents the integer representation of a signal frequency.

        #include <frequency.hpp>

        function divider

        inline constexpr int_t divider(
    frequency p_target
) const

        Calculate the frequency divider required to generate the output frequency provided.

        Parameters:

        • p_target - the target output frequency

        Return: constexpr int_t - the divider, when applied to this frequency, will achieve the p_target frequency. A value of is an error and 0 indicates that the output frequency is greater than this frequency and there does not exist an integer divider that can produce the output frequency.
