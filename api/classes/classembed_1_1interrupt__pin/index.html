        Detailed Description

        class embed::interrupt_pin;

        Digital interrupt pin hardware abstraction.

        Use this to trigger an interrupt service routine (ISR) when a pin detects a falling edge (when the pin’s voltage transitions from HIGH to LOW), a rising edge (when the pin’s voltage transitions from LOW to HIGH), or any transition of state on the pin.

        Public Types Documentation

        enum trigger_edge

        Enumerator Value Description
        falling 0 Trigger and interrupt when a pin transitions from HIGH voltage to LOW voltage.
        rising 1 Trigger and interrupt when a pin transitions from LOW voltage to HIGH voltage.
        both 2 Trigger and interrupt when a pin transitions it state.

        The condition in which an interrupt is triggered.
