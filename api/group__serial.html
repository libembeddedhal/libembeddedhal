size_t  p_length 
)
inlinenoexcept

Delay execution until the serial buffer has reached a specific number of buffered bytes.

NOTE: If the length is greater than the serial port's capacity this will loop forever.

Parameters
p_serial- serial port to wait for
p_length- the number of bytes that need to be buffered before this function returns.
Returns
boost::leaf::result<void> - return an error if a call to serial::bytes_available returns an error from the serial port.
