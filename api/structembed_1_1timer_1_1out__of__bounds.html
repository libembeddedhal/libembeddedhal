Detailed Description

Error type indicating that the desired time delay is not achievable with this timer.

Usually this occurs if the time delay is too small or too big based on what is possible with the driving frequency of the timer and along with any pre-scalars before the counting register.

How to handle these errors:

  • In cases where the program is scanning for the fastest delay, this is to be expected. The error will report the minimum possible delay which can then be assigned. Same for the maximum possible delay.
  • In most other cases, this is usually a bug in the code and cannot be handled in code and should be treated as such. Drivers using a timer, if they need an exact number will not be usable with this timer if it throws this error, which will either require another timer that can perform this work be used or increasing the clock rate fed into the timer in order to increase its frequency range.
