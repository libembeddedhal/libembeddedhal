Detailed Description

template<std::floating_point float_t = float>
class hal::percentage< float_t >

A type representing a percentage value.

This class guarantees that its value is always [-1.0, 1.0], thus bounds checking is never required when retrieving its internal floating point value either via value(), static_cast<float> or static_cast<double>
